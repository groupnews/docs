# Newsletters in GroupNews

Newsletters are centralized sources of shared information within groups news. Groups can set up as many newsletters as they need. Newsletters are given a name and description and can be given different [Visibility Settings](https://docs.groupnews.com/using-groupnews/newsletters/settings/ "Settings").
